# Aweil School of Management & Business Studies Website

## Overview
This is the official website for **Aweil School of Management & Business Studies**, designed for South Sudan. It showcases the school's programs, admissions, contact info, and provides a polished user experience with a responsive purple/gold theme.

## Structure
- `index.html` — Home page with hero banner and icons
- `about.html` — About page, mission & vision, learning & leadership
- `programs.html` — Programs overview (Certificate, Diploma, Professional)
- `admission.html` — Online admission form
- `thank-you.html` — Submission success page
- `contact.html` — Contact form and Google Maps integration
- `style.css` — Styling, layout, responsive design
- `menu.js` — Responsive menu toggle
- `images/` — All image assets used
- `favicon.ico` — Site favicon

## Technologies
- HTML5, CSS3
- Vanilla JavaScript
- Responsive Grid Layout
- SEO-friendly metadata
